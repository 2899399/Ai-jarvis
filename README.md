# 🤖 AI JARVIS — Personal Hindi AI Assistant

A fully local, private Hindi AI Assistant powered by LLaMA3 via Ollama. No API key needed, no data shared, 100% yours!

## 🚀 Features

- 💬 Hindi mein baat karta hai
- 🧠 Smart AI responses (LLaMA3)
- 🎤 Voice + Text dono modes
- 📂 Apps kholta hai (Chrome, Notepad, VS Code etc.)
- 🌐 Websites kholta hai (YouTube, GitHub etc.)
- 🔍 Google search karta hai
- 🕐 Time & Date batata hai
- 🔒 100% Local & Private — koi data bahar nahi jaata

## 🛠️ Requirements

```bash
pip install requests pyttsx3 speechrecognition pyaudio
```

Also install [Ollama](https://ollama.com) and run:
```bash
ollama pull llama3
```

## ▶️ How to Run

First start Ollama in CMD:
```bash
ollama serve
```

Then run JARVIS in CMD:
```bash
python ai_jarvis.py
```
